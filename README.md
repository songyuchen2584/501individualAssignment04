# Android Compose Assignment

## Q1 Gyroscope-Controlled Ball Game
- Uses `Sensor.TYPE_GYROSCOPE`
- Ball drawn with Compose `Canvas`
- Includes walls, obstacles, and a goal area

## Q2 Location Information
- Requests location permission
- Shows Google Map centered on user location
- Adds marker for current location
- Displays address
- Adds custom markers on map tap

## Q3 Polyline and Polygon
- Displays hiking trail with `Polyline`
- Displays park area with `Polygon`
- Lets user change width and color
- Overlay tap shows info
